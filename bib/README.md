# Bibliography Folder

Use this folder to store BibTeX files for easier citation management in the survey writing stage.

Suggested structure:

- `tta.bib`: TTA papers
- `ctta.bib`: CTTA papers
- `misc.bib`: related references, benchmarks, and surveys

Current example:

- `tta.bib`: `dong2025certaintta` (CertainTTA, Information Fusion 2025)
